# Улучшение качества голоса TTS

## Проблема
Голос звучит механически и затянуто.

## Решение

### 1. Исправлены параметры скорости
- Было: `rate=170` (неправильно для COM)
- Стало: `rate=2` (правильный диапазон -10 до 10 для COM)
- Скорость 2 делает речь немного быстрее и более живой

### 2. Улучшен выбор голоса
- Добавлен поиск более качественных голосов (Neural, Premium)
- Приоритет выбора: Neural > Premium > Irina > другие русские голоса

### 3. Настройки по умолчанию
- Скорость: 2 (немного ускоренная для более живого звука)
- Громкость: 100% (максимальная)

## Как настроить самостоятельно

### Изменение скорости
В файле `src/kim_voice/main.py` найдите строку:
```python
voice = KimVoice(rate=2, volume=100)
```

Параметры скорости для COM:
- `-10` — очень медленно
- `0` — нормальная скорость
- `2-4` — немного быстрее (рекомендуется для более живого звука)
- `5-7` — быстро
- `10` — очень быстро

### Установка более качественных голосов

Для более естественного звучания можно установить Neural голоса от Microsoft:

1. Откройте **Параметры Windows** → **Время и язык** → **Речь**
2. Установите дополнительные голоса (Neural голоса)
3. Перезапустите ассистента — код автоматически выберет лучший доступный голос

## Рекомендации

- Для более живого звука: используйте `rate=3-4`
- Для более естественного звука: установите Neural голоса от Microsoft
- Если голос слишком быстрый: уменьшите до `rate=1-2`
- Если голос слишком медленный: увеличьте до `rate=4-5`
