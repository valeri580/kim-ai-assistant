# Настройка голосового синтеза (TTS)

## Проблема

TTS технически работает (движок инициализирован, устройство найдено), но звука нет, хотя YouTube и другие приложения работают со звуком через Bluetooth колонку.

## Причина

Windows Speech API (SAPI) использует **устройство по умолчанию Windows**, а не то, которое устанавливается программно.

## Решение

### Шаг 1: Установите Bluetooth колонку как устройство по умолчанию

1. Откройте **Параметры Windows**:
   - Нажмите `Win + I`
   - Или через меню "Пуск" → Параметры

2. Перейдите в раздел звука:
   - **Система** → **Звук**
   - Или нажмите правой кнопкой на иконку динамика в трее → **Открыть параметры звука**

3. Выберите устройство вывода:
   - В разделе **"Выбор устройства вывода"**
   - Выберите **"Headphones (SberBoom Home 1205)"**

4. Убедитесь, что это устройство активно и громкость не на минимуме

### Шаг 2: Проверьте работу TTS

После установки устройства по умолчанию, запустите тест:

```bash
python scripts/test_simple_voice.py
```

Вы должны услышать тестовое сообщение.

## После настройки

После того, как вы установите Bluetooth колонку как устройство по умолчанию, TTS будет работать автоматически, и все голосовые функции будут использовать это устройство.
